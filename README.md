# Employee Salary Analysis (Python & Pandas)

# Project Overview
This project analyzes employee salary data using Python.  
The goal is to understand salary distribution, department-wise averages, and identify meaningful insights for business decision-making.

# Tools & Libraries Used
* Python
* Pandas
* Matplotlib

# Key Analysis Performed
* Calculated average salary per department
* Identified employees earning above their department average
* Created department-level salary summaries
* Exported analysis results to CSV files
* Visualized average salary per department

# Average Salary per Department

# Files Included
* employees.csv → Raw employee data
* employee_salary_analysis.csv → Cleaned and analyzed employee data
* department_summary.csv → Department-level salary summary
* salary_extremes.csv → Highest and lowest salary departments
* avg_salary_per_department.png → Salary visualization chart

# Conclusion
This analysis helps identify salary trends across departments and supports data-driven decision-making.  
The project demonstrates practical use of Python for real-world data analysis tasks.
